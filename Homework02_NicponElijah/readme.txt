Hey! Welcome to bad Pong.

upon loading, the game will start immedietly.

player 1 is blue on the right, and player 2 is red on the left.

THESE CONTROLS ARE OPTIMIZED FOR DEFAULT KEYBOARD SETTINGS ON THE mGBA EMULATOR 
Player 1: A -- paddle up
          Z -- paddle down
Player 2: ↑ -- paddle up (up key)
          ↓ -- paddle down (down key)

first to 3 wins.

at the end of the game, the winning player's color is shown. press start to reset and play again.


enjoy!

note: before I multiplied the speed by 2 for the newly spawned balls, the game was really boring (all balls would share the same x coordinate) so I changed it.
